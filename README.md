Objetivo
Explicar a importância da tecnologia no campo.

Apresentar exemplos de tecnologias utilizadas na agricultura.
Informar os benefícios da inovação para produtores e consumidores.

Conteúdo do site
O site aborda temas como:
Agricultura de precisão.
Uso de drones e sensores.
Máquinas agrícolas modernas.
Irrigação inteligente.
Sustentabilidade no campo.
Tecnologias utilizadas

HTML
CSS
JavaScript

Como executar
Baixe ou clone o projeto.
Abra a pasta do projeto.
Execute o arquivo index.html em qualquer navegador.

Público-alvo
O site é destinado a estudantes, professores e qualquer pessoa interessada em conhecer mais sobre as tecnologias utilizadas no setor agrícola.
